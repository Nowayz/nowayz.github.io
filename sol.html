Sol Framework

Features
General
  • Sol Framework, an internally multi-threaded LuaJIT based webscraping framework, features a collection of functions and libraries enabling the automation of any task on the internet.
  • The Sol Framework is an enterprise solution, and embodies quality, transparency, and efficiency above all else.
Easy to Learn
  • The web isn't complicated, yet most web-scraping tools hide the mechanics of whats actually happening, and provide restrictive (and slow) API calls.
  • Sol Framework is extremely fast and provides direct access to the variables that make up an HTTP request.
Major Features
  • Windows Support
  • Linux Support
  • Full API compatability between Win/Linux
  • Fast Startup / Low Memory Footprint
  • IPv4 Support
  • IPv6 Support
  • Multi IP-Address Support
  • Customizable DNS Resolver
  • DNS Lookup Priority (A/AAAA Record)
  • DNS Lookup Fallback
  • Asynchronous / Synchronous API
  • Internally Multithreaded (No manual threading)
  • High Performance Optimized
  • All major databases supported (through LuaJIT FFI)
  • Javascript Object Notation (JSON) Support
  • Hand-Written API Reference (No auto-generated 'documentation')
  • Integrate very easily with other console applications
  • gzip compression API
  • base64 encoding API
  • RFC-Compliant URL API
