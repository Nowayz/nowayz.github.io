Access Hook
Written in C for x86-64, Access Hook is a WinAPI powered hook library allowing function hooking without overwriting any instructions or pointers.

Implemented through CPU page access violations and a Vectored Exception Handler, Access Hook can provide a stealthy way to hook binaries which may be performing runtime checksums without worrying about hiding code modifications.
